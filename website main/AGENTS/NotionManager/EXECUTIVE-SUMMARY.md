## 🚀 Production Usage

### Quick Start

```bash
# Run comprehensive tests
node test-v3.6-comprehensive.js
# Expected: 24/24 tests passing

# Run production demo
node production-sync.js
# Expected: All validations pass, demo completes

# Actual production sync
node production-sync.js --sync
# Expected: Files synced to Notion successfully
```

### Programmatic Usage

```javascript
import { SyncManagerV2 } from './sync-with-validation-v2.js';

const manager = new SyncManagerV2({ verbose: true });

// Sync single file
const result = await manager.syncFile('document.md', {
  title: 'My Document',
  showProgress: true,
});

// Sync multiple files
const results = await manager.syncFiles([
  'doc1.md',
  'doc2.md',
  'doc3.md',
], { showProgress: true });
```
