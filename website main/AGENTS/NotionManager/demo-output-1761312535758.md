Our API uses OAuth 2.0 for authentication. Here's the flow:

![OAuth 2.0 authentication sequence diagram_](https://images.unsplash.com/photo-1558494949-ef010cbdcc31?w=800)

**Steps**:

1. User initiates login

1. Redirected to authorization server

1. User grants permissions

1. Application receives access token

1. API calls made with token
