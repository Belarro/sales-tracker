# Protocol Enforcement Solution: Agent Formatting & Terminal Titles

## Overview

This document outlines an implementation plan to ensure consistent enforcement of:
1. Agent response formatting (prefix/suffix with agent names)
2. Terminal window title updates

The solution focuses on creating reliable enforcement mechanisms that work regardless of activation method or context.

Update `CLAUDE.md` with a dedicated Protocol Enforcement section that includes:

```markdown
## Protocol Enforcement

CRITICAL DIRECTIVE: The following formatting rules MUST be followed by ALL agents without exception:

1. **Response Format Requirement**:
   - EVERY agent response MUST begin with: `[AGENT_NAME]: `
   - EVERY agent response MUST end with: `-- [AGENT_NAME]`

2. **Terminal Title Requirement**:
   - Upon activation, EVERY agent MUST update the terminal title using the format:
```

### 4. Post-Response Validation Layer

Create a validation script that can be integrated into CI to check for protocol compliance:

```python
#!/usr/bin/env python3
# protocol_validator.py - Validates agent responses for protocol compliance

import sys
import re
import os

def validate_response(text, agent_name):
    # Check for proper prefix
    prefix_pattern = rf"^\[{agent_name}\]: "
    prefix_match = re.search(prefix_pattern, text, re.MULTILINE)
    
    # Check for proper suffix
    suffix_pattern = rf"-- \[{agent_name}\]$"
    suffix_match = re.search(suffix_pattern, text, re.MULTILINE)
    
    if not prefix_match:
        print(f"ERROR: Missing required prefix: [{agent_name}]: ")
        return False
        
    if not suffix_match:
        print(f"ERROR: Missing required suffix: -- [{agent_name}]")
        return False
        
    return True

def main():
    if len(sys.argv) < 3:
        print("Usage: protocol_validator.py <agent_name> <response_text_file>")
        sys.exit(1)
        
    agent_name = sys.argv[1]
    response_file = sys.argv[2]
    
    with open(response_file, 'r') as f:
        response_text = f.read()
        
    if validate_response(response_text, agent_name):
        print("Response format complies with protocol standards.")
        sys.exit(0)
    else:
        sys.exit(1)

if __name__ == "__main__":
    main()
```
