# Volcano Agent Toolkit

This directory contains specialized tools and resources for the Volcano agent's Chasm framework application implementation work.

## Directory Structure

- `templates/` - Chasm framework application templates and boilerplates
- `patterns/` - Common implementation patterns and best practices
- `examples/` - Example implementations and reference applications
- `tools/` - Specialized development and testing tools
- `docs/` - Framework-specific documentation and guides

## Usage

The Volcano agent will prioritize resources in this toolkit directory before checking parent repositories, allowing for specialized Chasm framework-specific implementations and optimizations.

## Framework Focus

All tools and resources in this toolkit are optimized for:
- Chasm framework architecture patterns
- Prompt-based application development
- Performance optimization techniques
- Testing strategies specific to Chasm applications
- Documentation standards for framework implementations
