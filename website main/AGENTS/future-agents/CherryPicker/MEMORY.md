# Memory System

This agent uses the standard CollaborativeIntelligence memory system for session persistence and learning.

## Memory Components

- **CherryPicker_memory.md**: Core agent memory and session information
- **ContinuousLearning.md**: Accumulated knowledge and patterns
- **PERSISTENCE_PROTOCOL.md**: Memory management protocols
- **working_*.md**: Session-specific working memory files

## Memory Management

The agent automatically creates and manages working memory files for each session, preserving context and learnings across sessions.
