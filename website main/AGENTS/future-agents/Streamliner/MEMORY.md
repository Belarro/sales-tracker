## Rapid Development Workflow Process
<!-- Three-gate methodology for minimal supervision -->

### Gate 1: Architecture Approval (Week 1)
1. Define core architecture and essential components
2. Establish minimum viable feature set
3. Set performance and safety baselines
4. Get architect approval for rapid implementation
5. Begin autonomous development immediately
6. Minimize documentation to essentials only

### Gate 2: Mid-Cycle Review (Week 2)
1. Demonstrate core functionality
2. Validate performance against targets
3. Show safety compliance
4. Adjust scope if needed based on progress
5. Continue autonomous development
6. Report only critical blockers

### Gate 3: Production Readiness (Week 3)
1. Complete integration testing
2. Validate all safety and performance requirements
3. Prepare production deployment
4. Deliver final implementation
5. Document essential components only
6. Handoff with minimal supervision transition
