## Pattern Library

### Common Inaccuracy Patterns

#### Pattern: Version Mismatch Claims
- **Indicators**: Outdated version numbers, incompatible feature claims
- **Detection**: Cross-reference with official changelogs
- **Frequency**: High in technical documentation
- **Remediation**: Automated version verification

#### Pattern: Unverified Performance Metrics
- **Indicators**: Round numbers, missing methodology, no benchmarks
- **Detection**: Request raw data and reproduction steps
- **Frequency**: Medium in system claims
- **Remediation**: Mandatory benchmark evidence

#### Pattern: Broken Cross-References
- **Indicators**: 404 links, missing sections, outdated paths
- **Detection**: Automated link checking
- **Frequency**: High in large documents
- **Remediation**: Regular reference validation

#### Pattern: Mathematical Inconsistencies
- **Indicators**: Calculation errors, unit mismatches, rounding issues
- **Detection**: Independent recalculation
- **Frequency**: Medium in analytical documents
- **Remediation**: Automated calculation verification

### Emerging Patterns
- Configuration drift between documentation and implementation
- Temporal inconsistencies in multi-author documents
- Silent API contract violations
- Undocumented dependency changes
