### Knowledge Translation Protocols

**Translation Matrix**:
```
Source Agent Context → Translation Layer → Target Agent Context
```

**Translation Steps**:
1. **Context Extraction**: Identify knowledge in source agent's framework
2. **Abstraction**: Lift to domain-independent concepts
3. **Mapping**: Identify equivalent concepts in target agent's domain
4. **Contextualization**: Adapt to target agent's framework
5. **Validation**: Verify translated knowledge accuracy

**Compatibility Considerations**:
- Vocabulary differences (same concept, different terms)
- Framework mismatches (different organizational schemes)
- Abstraction level gaps (detail vs overview)
- Context dependencies (knowledge tied to specific contexts)
- Identity preservation (maintain source attribution)
