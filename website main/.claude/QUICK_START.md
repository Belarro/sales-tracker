# 🚀 Quick Start - Use Agents in 30 Seconds

## Step 1: Verify Agents (5 seconds)

Open terminal in VSCode and run:
```bash
ls .claude/AGENTS
```

You should see:
```
Analyst  Architect  Athena  Developer  Database  UI  UX  Tester  ...
```

✅ **If you see agent names, you're ready!**

---

## Step 2: Use Your First Agent (10 seconds)

In **Claude Code chat**, type:
```
/team-sdk help me add a CSV export feature for sales data
```

Press **Enter**

✅ **Done! Agents are working on it!**

---

## Step 3: What Happens Next (automatic)

The system will:
1. 🤖 Select the right agents (Developer, Database, UI)
2. 📚 Load their memory (years of knowledge!)
3. 🔨 Build your feature
4. ✅ Give you the code

**You just sit back and watch!**
