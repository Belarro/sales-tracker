## PARALYSIS DETECTION SYSTEM

### **Stagnation Indicators**
```c
typedef struct {
    uint32_t debate_duration_minutes;
    uint8_t consensus_level_percentage; 
    bool forward_progress_detected;
    uint16_t mission_critical_decisions_pending;
    uint32_t time_since_last_inclusion;
} ParalysisDetector;

// CRITICAL THRESHOLDS
#define MAX_DEBATE_TIME 15          // minutes
#define MIN_CONSENSUS_REQUIRED 67   // percentage  
#define MAX_STAGNATION_PERIOD 1440  // minutes (24 hours)
#define EMERGENCY_ACTIVATION_THRESHOLD 5  // pending critical decisions
```

### **Automatic Paralysis Detection**
```c
bool detect_mission_paralysis(ParalysisDetector status) {
    return (
        status.debate_duration_minutes > MAX_DEBATE_TIME ||
        status.consensus_level_percentage < MIN_CONSENSUS_REQUIRED ||
        !status.forward_progress_detected ||
        status.mission_critical_decisions_pending > EMERGENCY_ACTIVATION_THRESHOLD ||
        status.time_since_last_inclusion > MAX_STAGNATION_PERIOD
    );
}
```
