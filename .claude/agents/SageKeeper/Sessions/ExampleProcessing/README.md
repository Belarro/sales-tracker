## Sample User Statement

For this example, we will process the following statement:

> "I've noticed that when we load large projects in Claude Code, memory loading is much more efficient if we segment the memory files and load them based on the current context rather than loading everything at once. This approach seems to reduce token usage by about 30-40% while maintaining all the critical functionality."
