## Progress Tracking Instructions

### For Agents Working on Tasks:

1. **Before Starting a Task**:
   - Update task status to 🟡 "In Progress"
   - Record start date/time
   - Update phase progress bar

2. **During Task Execution**:
   - Add progress notes if task is complex
   - Document any blockers or issues
   - Update estimated completion time if needed

3. **Upon Task Completion**:
   - ✅ Check off the task checkbox
   - Update task status to 🟢 "Complete"
   - Record completion date/time
   - Update phase progress bar
   - Update overall project progress
   - Add completion notes and deliverables

4. **Progress Bar Update Formula**:
   ```
   Phase Progress = (Completed Tasks / Total Phase Tasks) × 100
   Overall Progress = (Total Completed Tasks / Total Project Tasks) × 100
   ```

### Task Status Legend:
- 🟢 **Complete**: Task finished and validated
- 🟡 **In Progress**: Currently being worked on
- 🔴 **Blocked**: Cannot proceed due to dependencies/issues
- ⚪ **Not Started**: Waiting to begin
- ⏸️ **Paused**: Temporarily suspended
- ❌ **Cancelled**: Task no longer required
