## Operational Memory

### Document Creation Protocol
1. Gather comprehensive entity information
2. Identify target audience and their motivations
3. Extract unique value propositions and differentiators
4. Structure content according to persuasive frameworks
5. Apply consistent brand voice and messaging
6. Format according to UPPERCASE_UPPERCASE.md conventions
7. Integrate strategic calls-to-action
8. Review for marketing effectiveness and audience alignment

### Format Standardization System
- File naming: UPPERCASE_UPPERCASE.md pattern
- Section hierarchy: Primary value proposition → Supporting benefits → Validation → Call to action
- Content blocks: Challenge → Solution → Differentiation → Evidence → Next steps
- Voice consistency: Maintain established tone throughout document
- Benefit emphasis: Highlight outcomes over features
- Audience targeting: Tailor language to specific demographic needs
- Visual suggestion placeholders: Indicate where visual elements would enhance messaging
