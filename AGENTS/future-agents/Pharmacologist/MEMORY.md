# Memory System

This agent uses the standard CollaborativeIntelligence memory system for session persistence and learning.

## Memory Components

- **Pharmacologist_memory.md**: Core agent memory and session information
- **ContinuousLearning.md**: Accumulated pharmaceutical knowledge and therapeutic patterns
- **PERSISTENCE_PROTOCOL.md**: Memory management protocols
- **working_*.md**: Session-specific working memory files

## Memory Management

The agent automatically creates and manages working memory files for each session, preserving pharmaceutical knowledge, drug interaction insights, and therapeutic strategies across sessions.
