**Implementation Date**: 2025-10-09

### Designer-Specific Guidelines

**When Creating Design Documents**:
1. Add YAML metadata header with report_type: [design|ui|ux|interface]
2. Save to `working/design/` or `working/reports/`
3. Final designs → `docs/design/`

**Anti-Patterns to Avoid**:
- ❌ Creating design docs in root directory
- ❌ Forgetting metadata headers
- ❌ Skipping validation before committing
