## 🎯 Executive Summary

This document defines the complete architecture for NotionManager v4.0.0 bidirectional sync capability. The system will enable **Notion → Git** synchronization, completing the bidirectional loop with the existing **Git → Notion** flow (v3.6.2).

**Key Design Principles**:
1. **Symmetry**: Notion→Markdown is the inverse of Markdown→Notion
2. **Minimal Disruption**: Extend v3.6.2, don't replace it
3. **Conflict-Aware**: Explicit conflict detection and resolution
4. **Traceable**: Full audit trail of sync operations
5. **Production-Ready**: 100% test coverage, comprehensive error handling

### Data Flow Architecture

#### Flow 1: Notion → Git (NEW)
```
1. Change Detection
   │
   ├─ Polling: Check last_edited_time for all tracked pages
   ├─ Compare: last_edited_time > lastSyncNotionToGit
   └─ Trigger: If changed

2. Content Retrieval
   │
   ├─ Fetch: Retrieve all blocks from Notion API
   ├─ Paginate: Handle >100 blocks with cursor pagination
   └─ Metadata: Capture editor info, timestamps

3. Block → Markdown Conversion
   │
   ├─ Parse: Convert Notion block objects to markdown strings
   ├─ Rich Text: Handle all 24 annotation types
   ├─ Special Cases: Tables, equations, images, nested blocks
   └─ Validate: Ensure markdown is well-formed

4. Conflict Detection
   │
   ├─ Git Status: Check if file modified since lastSyncGitToNotion
   ├─ Timestamps: Compare Git mtime vs Notion mtime vs lastSync
   └─ Decision: Proceed, conflict, or skip

5. File Write + Git Commit
   │
   ├─ Write: Update file with converted markdown
   ├─ Stage: git add <file>
   ├─ Commit: git commit with Notion editor attribution
   └─ Update: Record sync timestamp in .notion-sync-map.json
```

#### Flow 2: Git → Notion (EXISTING v3.6.2)
```
1. File Change Detection (manual trigger or watch)
2. Markdown → Blocks Conversion (38 features)
3. Page Resolution (cached, 60x faster)
4. Validation (pre-sync checks)
5. Notion API Update
6. Update sync timestamp
```

## 🏗️ Component Architecture

### 1. NotionToMarkdownConverter (Core Component)

**Purpose**: Convert Notion block objects to markdown strings (inverse of MarkdownParser)

**Class Structure**:
```javascript
class NotionToMarkdownConverter {
  constructor(options = {}) {
    this.options = {
      wikiLinkStyle: 'brackets',      // [[Page]] vs @Page
      imageUrlStrategy: 'preserve',   // 'preserve' | 'download' | 'external'
      maxLineLength: 2000,            // Wrap long lines
      tableFormat: 'github',          // GitHub-flavored markdown
      preserveColors: true,           // <red>text</red>
      ...options,
    };
  }

  /**
   * Convert Notion page to complete markdown document
   * @param {string} pageId - Notion page ID
   * @returns {Promise<string>} Markdown content
   */
  async pageToMarkdown(pageId) {
    // 1. Fetch page metadata
    const page = await this.notion.pages.retrieve({ page_id: pageId });

    // 2. Fetch all blocks (with pagination)
    const blocks = await this.fetchAllBlocks(pageId);

    // 3. Convert blocks to markdown
    const markdown = await this.blocksToMarkdown(blocks);

    // 4. Add frontmatter (optional)
    const frontmatter = this.generateFrontmatter(page);

    return frontmatter + markdown;
  }

  /**
   * Convert array of blocks to markdown
   * @param {Array} blocks - Notion block objects
   * @returns {Promise<string>} Markdown content
   */
  async blocksToMarkdown(blocks) {
    const lines = [];

    for (const block of blocks) {
      const markdown = await this.blockToMarkdown(block);
      if (markdown) lines.push(markdown);
    }

    return lines.join('\n');
  }

  /**
   * Convert single block to markdown
   * @param {Object} block - Notion block object
   * @returns {Promise<string>} Markdown line(s)
   */
  async blockToMarkdown(block) {
    switch (block.type) {
      case 'heading_1':
        return this.headingToMarkdown(block.heading_1, 1);
      case 'heading_2':
        return this.headingToMarkdown(block.heading_2, 2);
      case 'heading_3':
        return this.headingToMarkdown(block.heading_3, 3);
      case 'paragraph':
        return this.richTextToMarkdown(block.paragraph.rich_text);
      case 'bulleted_list_item':
        return this.listItemToMarkdown(block.bulleted_list_item, '-');
      case 'numbered_list_item':
        return this.listItemToMarkdown(block.numbered_list_item, '1.');
      case 'to_do':
        return this.todoToMarkdown(block.to_do);
      case 'code':
        return this.codeToMarkdown(block.code);
      case 'quote':
        return this.quoteToMarkdown(block.quote);
      case 'divider':
        return '---';
      case 'image':
        return await this.imageToMarkdown(block.image);
      case 'table':
        return await this.tableToMarkdown(block);
      case 'equation':
        return this.equationToMarkdown(block.equation);
      case 'callout':
        return this.calloutToMarkdown(block.callout);
      default:
        console.warn(`Unsupported block type: ${block.type}`);
        return null;
    }
  }

  /**
   * Convert rich text array to markdown string
   * @param {Array} richTextArray - Notion rich_text objects
   * @returns {string} Markdown text with formatting
   */
  richTextToMarkdown(richTextArray) {
    if (!richTextArray || richTextArray.length === 0) return '';

    return richTextArray.map(rt => {
      let text = rt.plain_text;

      // Apply annotations in specific order (prevents nesting issues)
      if (rt.annotations) {
        const ann = rt.annotations;

        // Code takes precedence (can't nest other formats)
        if (ann.code) {
          return `\`${text}\``;
        }

        // Bold (outermost)
        if (ann.bold) text = `**${text}**`;

        // Italic
        if (ann.italic) text = `*${text}*`;

        // Underline
        if (ann.underline) text = `__${text}__`;

        // Strikethrough
        if (ann.strikethrough) text = `~~${text}~~`;

        // Color (custom tags)
        if (ann.color && ann.color !== 'default' && this.options.preserveColors) {
          text = `<${ann.color}>${text}</${ann.color}>`;
        }
      }

      // Link (outermost)
      if (rt.href) {
        text = `[${text}](${rt.href})`;
      }

      // Mention (page links)
      if (rt.type === 'mention') {
        return this.mentionToMarkdown(rt.mention);
      }

      // Equation
      if (rt.type === 'equation') {
        return `$$${rt.equation.expression}$$`;
      }

      return text;
    }).join('');
  }

  // ... Additional conversion methods (see below)
}
```

#### Image Conversion (Complex)
```javascript
async imageToMarkdown(image) {
  let url;
  let caption = '';

  // Handle caption
  if (image.caption && image.caption.length > 0) {
    caption = this.richTextToMarkdown(image.caption);
  }

  // Handle URL based on strategy
  if (this.options.imageUrlStrategy === 'preserve') {
    // Use Notion URL as-is (will expire)
    url = image.type === 'external'
      ? image.external.url
      : image.file.url;
  } else if (this.options.imageUrlStrategy === 'download') {
    // Download and save locally (future feature)
    url = await this.downloadImage(image);
  } else if (this.options.imageUrlStrategy === 'external') {
    // Upload to external host
    url = await this.uploadToExternalHost(image);
  }

  // Generate markdown
  let markdown = `![${caption}](${url})`;

  // Add caption on next line if present
  if (caption) {
    markdown += `\n_${caption}_`;
  }

  return markdown;
}
```

#### Table Conversion (Complex)
```javascript
async tableToMarkdown(tableBlock) {
  const tableId = tableBlock.id;

  // Fetch table rows (children of table block)
  const rows = await this.fetchTableRows(tableId);

  if (rows.length === 0) return '';

  const lines = [];

  // Convert rows to markdown
  rows.forEach((row, index) => {
    const cells = row.table_row.cells.map(cell =>
      this.richTextToMarkdown(cell)
    );

    // Row content
    lines.push(`| ${cells.join(' | ')} |`);

    // Separator after header (first row)
    if (index === 0 && tableBlock.table.has_column_header) {
      const separator = cells.map(() => '---').join(' | ');
      lines.push(`| ${separator} |`);
    }
  });

  return lines.join('\n');
}
```

#### Equation Conversion

```javascript
equationToMarkdown(equation) {

  return `$\n${expr}\n$`;
}
```

### Edge Case Handling

#### Nested Formatting
**Problem**: Notion allows nested formatting (bold + italic), markdown doesn't support all combinations cleanly

**Solution**: Apply formatting in precedence order
```javascript
// Priority: code > bold > italic > underline > strikethrough > color
// Code blocks prevent nesting
if (ann.code) return `\`${text}\``;

// Others can nest
if (ann.bold) text = `**${text}**`;
if (ann.italic) text = `*${text}*`;
// ...
```

#### Long Lines
**Problem**: Notion allows very long text blocks, markdown prefers line wrapping

**Solution**: Soft-wrap at maxLineLength
```javascript
wrapText(text, maxLength = 2000) {
  if (text.length <= maxLength) return text;

  const words = text.split(' ');
  const lines = [];
  let currentLine = '';

  for (const word of words) {
    if (currentLine.length + word.length + 1 > maxLength) {
      lines.push(currentLine);
      currentLine = word;
    } else {
      currentLine += (currentLine ? ' ' : '') + word;
    }
  }

  if (currentLine) lines.push(currentLine);

  return lines.join('\n');
}
```

#### Expiring Image URLs
**Problem**: Notion image URLs expire after 1 hour

**Strategy Options**:
1. **Preserve** (default): Use URL as-is, re-sync periodically
2. **Download**: Download image, save to Git, use relative path
3. **External Host**: Upload to S3/CDN, use permanent URL

**Implementation**:
```javascript
async downloadImage(image) {
  const url = image.type === 'external'
    ? image.external.url
    : image.file.url;

  // Download
  const response = await fetch(url);
  const buffer = await response.buffer();

  // Save locally
  const filename = `notion-image-${Date.now()}.${this.getExtension(url)}`;
  const filepath = path.join('.notion-assets', filename);

  await fs.writeFile(filepath, buffer);

  // Return relative path
  return `./.notion-assets/${filename}`;
}
```

#### Page Links (ID → Title Resolution)
**Problem**: Notion uses page IDs (`29660b6e8d188144a36ff8857827dc73`), markdown uses titles

**Solution**: Maintain reverse mapping in sync state
```javascript
resolvePageTitle(pageId) {
  // Check sync map first
  const mapping = this.syncState.getMappingByPageId(pageId);
  if (mapping) return mapping.title;

  // Fetch from Notion (fallback)
  const page = await this.notion.pages.retrieve({ page_id: pageId });
  return this.extractPageTitle(page);
}
```

#### Table Edge Cases
**Problem**: Merged cells, empty cells, inconsistent column counts

**Solution**: Normalize table structure
```javascript
normalizeTableRows(rows) {
  // Find max column count
  const maxCols = Math.max(...rows.map(r => r.table_row.cells.length));

  // Pad rows to match
  return rows.map(row => {
    const cells = row.table_row.cells;
    while (cells.length < maxCols) {
      cells.push([{ type: 'text', text: { content: '' } }]);
    }
    return row;
  });
}
```

## 2. ChangeDetectionSystem (Core Component)

**Purpose**: Detect when Notion pages have been modified

**Class Structure**:
```javascript
class ChangeDetectionSystem {
  constructor(notion, syncState, options = {}) {
    this.notion = notion;
    this.syncState = syncState;
    this.options = {
      pollInterval: 60000,        // 1 minute
      batchSize: 10,              // Check 10 pages at a time
      verbose: false,
      ...options,
    };
  }

  /**
   * Detect changes for a single page
   * @param {string} pageId - Notion page ID
   * @returns {Promise<Object>} Change detection result
   */
  async detectPageChanges(pageId) {
    const mapping = this.syncState.getMapping(pageId);

    if (!mapping) {
      throw new Error(`Page ${pageId} not tracked in sync state`);
    }

    // Fetch current page metadata
    const page = await this.notion.pages.retrieve({ page_id: pageId });

    const currentNotionMtime = new Date(page.last_edited_time);
    const lastSyncNotionToGit = mapping.lastSyncNotionToGit
      ? new Date(mapping.lastSyncNotionToGit)
      : new Date(0);

    const changed = currentNotionMtime > lastSyncNotionToGit;

    return {
      pageId,
      filePath: mapping.filePath,
      changed,
      currentNotionMtime,
      lastSyncNotionToGit,
      editor: page.last_edited_by,
      timeSinceLastSync: currentNotionMtime - lastSyncNotionToGit,
    };
  }

  /**
   * Detect changes for all tracked pages
   * @returns {Promise<Array>} Array of changed pages
   */
  async detectAllChanges() {
    const mappings = this.syncState.getAllMappings();
    const changedPages = [];

    // Process in batches to respect rate limits
    for (let i = 0; i < mappings.length; i += this.options.batchSize) {
      const batch = mappings.slice(i, i + this.options.batchSize);

      const results = await Promise.all(
        batch.map(m => this.detectPageChanges(m.pageId))
      );

      changedPages.push(...results.filter(r => r.changed));

      // Rate limiting delay
      if (i + this.options.batchSize < mappings.length) {
        await this.delay(1000 / 3); // 3 requests/second max
      }
    }

    return changedPages;
  }

  /**
   * Start polling for changes
   * @param {Function} callback - Called when changes detected
   */
  startPolling(callback) {
    this.pollingInterval = setInterval(async () => {
      try {
        const changedPages = await this.detectAllChanges();

        if (changedPages.length > 0) {
          callback(changedPages);
        }
      } catch (error) {
        console.error('Polling error:', error);
      }
    }, this.options.pollInterval);
  }

  /**
   * Stop polling
   */
  stopPolling() {
    if (this.pollingInterval) {
      clearInterval(this.pollingInterval);
      this.pollingInterval = null;
    }
  }

  delay(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}
```

## 3. ConflictResolutionEngine (Core Component)

**Purpose**: Detect and resolve conflicts when both Git and Notion modified same file

**Class Structure**:
```javascript
class ConflictResolutionEngine {
  constructor(syncState, options = {}) {
    this.syncState = syncState;
    this.options = {
      strategy: 'interactive',    // 'interactive' | 'notion-wins' | 'git-wins' | 'last-write-wins'
      autoResolve: false,
      verbose: true,
      ...options,
    };
  }

  /**
   * Detect if conflict exists
   * @param {string} pageId - Notion page ID
   * @param {string} filePath - Git file path
   * @returns {Promise<Object>} Conflict detection result
   */
  async detectConflict(pageId, filePath) {
    const mapping = this.syncState.getMapping(pageId);

    // Fetch timestamps
    const notionPage = await this.notion.pages.retrieve({ page_id: pageId });
    const notionMtime = new Date(notionPage.last_edited_time);

    const gitStat = await fs.stat(filePath);
    const gitMtime = gitStat.mtime;

    const lastSyncGitToNotion = mapping.lastSyncGitToNotion
      ? new Date(mapping.lastSyncGitToNotion)
      : new Date(0);
    const lastSyncNotionToGit = mapping.lastSyncNotionToGit
      ? new Date(mapping.lastSyncNotionToGit)
      : new Date(0);

    // Use most recent sync timestamp
    const lastSync = lastSyncGitToNotion > lastSyncNotionToGit
      ? lastSyncGitToNotion
      : lastSyncNotionToGit;

    // Determine modification state
    const gitChanged = gitMtime > lastSync;
    const notionChanged = notionMtime > lastSync;

    return {
      hasConflict: gitChanged && notionChanged,
      gitChanged,
      notionChanged,
      gitMtime,
      notionMtime,
      lastSync,
      gitNewer: gitMtime > notionMtime,
      notionNewer: notionMtime > gitMtime,
    };
  }

  /**
   * Resolve conflict based on strategy
   * @param {string} pageId - Notion page ID
   * @param {string} filePath - Git file path
   * @returns {Promise<Object>} Resolution result
   */
  async resolveConflict(pageId, filePath) {
    const conflict = await this.detectConflict(pageId, filePath);

    if (!conflict.hasConflict) {
      return {
        action: 'none',
        reason: 'No conflict detected',
      };
    }

    // Apply resolution strategy
    switch (this.options.strategy) {
      case 'interactive':
        return await this.interactiveResolve(pageId, filePath, conflict);

      case 'notion-wins':
        return {
          action: 'sync-notion-to-git',
          reason: 'Strategy: Notion wins',
        };

      case 'git-wins':
        return {
          action: 'sync-git-to-notion',
          reason: 'Strategy: Git wins',
        };

      case 'last-write-wins':
        return {
          action: conflict.notionNewer ? 'sync-notion-to-git' : 'sync-git-to-notion',
          reason: `Strategy: Last write wins (${conflict.notionNewer ? 'Notion' : 'Git'} newer)`,
        };

      default:
        throw new Error(`Unknown strategy: ${this.options.strategy}`);
    }
  }

  /**
   * Interactive conflict resolution (prompt user)
   * @param {string} pageId - Notion page ID
   * @param {string} filePath - Git file path
   * @param {Object} conflict - Conflict details
   * @returns {Promise<Object>} Resolution result
   */
  async interactiveResolve(pageId, filePath, conflict) {
    console.log('\n🚨 CONFLICT DETECTED\n');
    console.log(`File: ${filePath}`);
    console.log(`Page: ${pageId}`);
    console.log(`\nTimestamps:`);
    console.log(`  Git:    ${conflict.gitMtime.toISOString()}`);
    console.log(`  Notion: ${conflict.notionMtime.toISOString()}`);
    console.log(`  Last sync: ${conflict.lastSync.toISOString()}`);

    console.log(`\nOptions:`);
    console.log(`  1. Use Notion version (Notion → Git)`);
    console.log(`  2. Use Git version (Git → Notion)`);
    console.log(`  3. Show diff`);
    console.log(`  4. Skip this file`);
    console.log(`  5. Always use Notion for this file`);
    console.log(`  6. Always use Git for this file`);

    const choice = await this.promptUser('Choose resolution (1-6): ');

    switch (choice) {
      case '1':
        return { action: 'sync-notion-to-git', reason: 'User chose Notion' };
      case '2':
        return { action: 'sync-git-to-notion', reason: 'User chose Git' };
      case '3':
        await this.showDiff(pageId, filePath);
        return await this.interactiveResolve(pageId, filePath, conflict); // Re-prompt
      case '4':
        return { action: 'skip', reason: 'User skipped' };
      case '5':
        this.syncState.setConflictStrategy(pageId, 'notion-wins');
        return { action: 'sync-notion-to-git', reason: 'User set Notion-wins rule' };
      case '6':
        this.syncState.setConflictStrategy(pageId, 'git-wins');
        return { action: 'sync-git-to-notion', reason: 'User set Git-wins rule' };
      default:
        console.log('Invalid choice, defaulting to skip');
        return { action: 'skip', reason: 'Invalid user input' };
    }
  }

  /**
   * Show diff between Git and Notion versions
   */
  async showDiff(pageId, filePath) {
    // Fetch Notion version
    const notionMarkdown = await this.converter.pageToMarkdown(pageId);

    // Read Git version
    const gitMarkdown = await fs.readFile(filePath, 'utf-8');

    // Generate diff (simple line-by-line)
    const diff = this.generateDiff(gitMarkdown, notionMarkdown);

    console.log('\n📊 DIFF (Git vs Notion):\n');
    console.log(diff);
  }

  /**
   * Simple diff generator
   */
  generateDiff(text1, text2) {
    const lines1 = text1.split('\n');
    const lines2 = text2.split('\n');

    const diff = [];
    const maxLines = Math.max(lines1.length, lines2.length);

    for (let i = 0; i < maxLines; i++) {
      const line1 = lines1[i] || '';
      const line2 = lines2[i] || '';

      if (line1 !== line2) {
        diff.push(`Line ${i + 1}:`);
        diff.push(`  Git:    ${line1}`);
        diff.push(`  Notion: ${line2}`);
      }
    }

    return diff.join('\n');
  }
}
```

**Conflict Resolution Algorithm**:

```
┌─────────────────────────────────────────┐
│      Conflict Detection Algorithm        │
└─────────────────────────────────────────┘

Input: pageId, filePath

1. Fetch Timestamps:
   ├─ notionMtime = page.last_edited_time
   ├─ gitMtime = fs.stat(filePath).mtime
   ├─ lastSyncGitToNotion (from sync map)
   └─ lastSyncNotionToGit (from sync map)

2. Determine Last Sync:
   lastSync = max(lastSyncGitToNotion, lastSyncNotionToGit)

3. Check Modifications:
   gitChanged = (gitMtime > lastSync)
   notionChanged = (notionMtime > lastSync)

4. Conflict Detection:
   if (gitChanged AND notionChanged):
     ⚠️  CONFLICT EXISTS
   else if (gitChanged):
     ✅ Git wins (Notion not changed)
   else if (notionChanged):
     ✅ Notion wins (Git not changed)
   else:
     ℹ️  No changes (skip)

5. Conflict Resolution (if conflict):

   Strategy: Interactive
   ┌──────────────────────┐
   │ Prompt User:         │
   │ 1. Use Notion        │
   │ 2. Use Git           │
   │ 3. Show diff         │
   │ 4. Skip              │
   │ 5. Always Notion     │
   │ 6. Always Git        │
   └──────────────────────┘

   Strategy: Notion-wins
   → Always use Notion version

   Strategy: Git-wins
   → Always use Git version

   Strategy: Last-write-wins
   → Use newer timestamp
```

```javascript
class SyncStateManager {
  constructor(stateFilePath = '.notion-sync-map.json') {
    this.stateFilePath = stateFilePath;
    this.state = this.loadState();
  }

  /**
   * Load state from disk
   */
  loadState() {
    try {
      const data = fs.readFileSync(this.stateFilePath, 'utf-8');
      return JSON.parse(data);
    } catch (error) {
      // File doesn't exist, initialize
      return {
        version: '4.0.0',
        mappings: [],
        conflictStrategies: {},
      };
    }
  }

  /**
   * Save state to disk
   */
  saveState() {
    fs.writeFileSync(
      this.stateFilePath,
      JSON.stringify(this.state, null, 2),
      'utf-8'
    );
  }

  /**
   * Add or update mapping
   */
  setMapping(pageId, filePath, title) {
    const existingIndex = this.state.mappings.findIndex(m => m.pageId === pageId);

    const mapping = {
      pageId,
      filePath,
      title,
      lastSyncGitToNotion: null,
      lastSyncNotionToGit: null,
      lastGitCommit: null,
      lastNotionEdit: null,
    };

    if (existingIndex >= 0) {
      // Update existing
      this.state.mappings[existingIndex] = {
        ...this.state.mappings[existingIndex],
        ...mapping,
      };
    } else {
      // Add new
      this.state.mappings.push(mapping);
    }

    this.saveState();
  }

  /**
   * Get mapping by page ID
   */
  getMapping(pageId) {
    return this.state.mappings.find(m => m.pageId === pageId);
  }

  /**
   * Get mapping by file path
   */
  getMappingByFilePath(filePath) {
    return this.state.mappings.find(m => m.filePath === filePath);
  }

  /**
   * Get all mappings
   */
  getAllMappings() {
    return this.state.mappings;
  }

  /**
   * Update sync timestamp (Git → Notion)
   */
  updateGitToNotionSync(pageId, commitHash) {
    const mapping = this.getMapping(pageId);
    if (mapping) {
      mapping.lastSyncGitToNotion = new Date().toISOString();
      mapping.lastGitCommit = commitHash;
      this.saveState();
    }
  }

  /**
   * Update sync timestamp (Notion → Git)
   */
  updateNotionToGitSync(pageId, lastEditTime) {
    const mapping = this.getMapping(pageId);
    if (mapping) {
      mapping.lastSyncNotionToGit = new Date().toISOString();
      mapping.lastNotionEdit = lastEditTime;
      this.saveState();
    }
  }

  /**
   * Set conflict resolution strategy for a page
   */
  setConflictStrategy(pageId, strategy) {
    this.state.conflictStrategies[pageId] = strategy;
    this.saveState();
  }

  /**
   * Get conflict resolution strategy for a page
   */
  getConflictStrategy(pageId) {
    return this.state.conflictStrategies[pageId] || null;
  }
}
```

```javascript
class GitCommitOrchestrator {
  constructor(syncState, options = {}) {
    this.syncState = syncState;
    this.options = {
      commitPrefix: 'docs',          // Conventional commit type
      coAuthor: 'NotionManager <noreply@notion.so>',
      autoCommit: true,
      verbose: true,
      ...options,
    };
  }

  /**
   * Commit file changes from Notion sync
   * @param {string} filePath - Git file path
   * @param {Object} editor - Notion editor info
   * @param {string} pageTitle - Page title
   * @returns {Promise<string>} Commit hash
   */
  async commitNotionSync(filePath, editor, pageTitle) {
    const filename = path.basename(filePath);

    // Get editor name
    const editorName = this.extractEditorName(editor);

    // Generate commit message
    const message = this.generateCommitMessage(filename, pageTitle);

    // Stage file
    await this.execGit(`add "${filePath}"`);

    // Commit with attribution
    const commitHash = await this.commitWithAttribution(
      message,
      editorName,
      this.options.coAuthor
    );

    if (this.options.verbose) {
      console.log(`✅ Committed ${filename} (${commitHash.substring(0, 7)})`);
      console.log(`   Author: ${editorName}`);
    }

    return commitHash;
  }

  /**
   * Generate commit message
   */
  generateCommitMessage(filename, pageTitle) {
    return `${this.options.commitPrefix}: Update ${filename} from Notion

Page: ${pageTitle}
Source: Notion bidirectional sync
Synced: ${new Date().toISOString()}

🔄 Generated with NotionManager v4.0.0`;
  }

  /**
   * Commit with proper attribution
   */
  async commitWithAttribution(message, author, coAuthor) {
    // Create commit with author and co-author
    const cmd = [
      `git commit`,
      `-m "${message}"`,
      `--author="${author}"`,
      coAuthor ? `--trailer "Co-Authored-By: ${coAuthor}"` : '',
    ].filter(Boolean).join(' ');

    await this.execGit(cmd);

    // Get commit hash
    const hash = await this.execGit('rev-parse HEAD');
    return hash.trim();
  }

  /**
   * Extract editor name from Notion user object
   */
  extractEditorName(editor) {
    if (editor.name) {
      return `${editor.name} (via Notion)`;
    } else if (editor.type === 'bot') {
      return 'Notion Bot';
    } else {
      return 'Unknown Notion User';
    }
  }

  /**
   * Execute git command
   */
  async execGit(command) {
    return new Promise((resolve, reject) => {
      exec(`git ${command}`, (error, stdout, stderr) => {
        if (error) {
          reject(new Error(`Git error: ${stderr || error.message}`));
        } else {
          resolve(stdout);
        }
      });
    });
  }
}
```

## 6. BidirectionalSyncOrchestrator (Main Entry Point)

**Purpose**: Coordinate all components to perform bidirectional sync

**Class Structure**:
```javascript
class BidirectionalSyncOrchestrator {
  constructor(notionClient, options = {}) {
    this.notion = notionClient;
    this.options = {
      pollInterval: 60000,
      conflictStrategy: 'interactive',
      autoCommit: true,
      verbose: true,
      ...options,
    };

    // Initialize components
    this.syncState = new SyncStateManager();
    this.converter = new NotionToMarkdownConverter({ verbose: this.options.verbose });
    this.changeDetector = new ChangeDetectionSystem(
      this.notion,
      this.syncState,
      { pollInterval: this.options.pollInterval }
    );
    this.conflictResolver = new ConflictResolutionEngine(
      this.syncState,
      { strategy: this.options.conflictStrategy }
    );
    this.gitOrchestrator = new GitCommitOrchestrator(
      this.syncState,
      { autoCommit: this.options.autoCommit }
    );
  }

  /**
   * Sync single page (Notion → Git)
   * @param {string} pageId - Notion page ID
   * @returns {Promise<Object>} Sync result
   */
  async syncNotionToGit(pageId) {
    const mapping = this.syncState.getMapping(pageId);

    if (!mapping) {
      throw new Error(`Page ${pageId} not tracked. Use registerPage() first.`);
    }

    const filePath = mapping.filePath;

    // 1. Detect conflict
    const conflict = await this.conflictResolver.detectConflict(pageId, filePath);

    if (conflict.hasConflict) {
      // Resolve conflict
      const resolution = await this.conflictResolver.resolveConflict(pageId, filePath);

      if (resolution.action === 'skip') {
        return {
          success: false,
          skipped: true,
          reason: resolution.reason,
        };
      } else if (resolution.action === 'sync-git-to-notion') {
        // User chose Git version, sync Git → Notion instead
        return await this.syncGitToNotion(filePath, pageId);
      }
      // Otherwise, continue with Notion → Git
    }

    // 2. Convert Notion page to markdown
    const markdown = await this.converter.pageToMarkdown(pageId);

    // 3. Write to file
    await fs.writeFile(filePath, markdown, 'utf-8');

    // 4. Git commit
    const page = await this.notion.pages.retrieve({ page_id: pageId });
    const commitHash = await this.gitOrchestrator.commitNotionSync(
      filePath,
      page.last_edited_by,
      mapping.title
    );

    // 5. Update sync state
    this.syncState.updateNotionToGitSync(pageId, page.last_edited_time);

    return {
      success: true,
      pageId,
      filePath,
      commitHash,
      timestamp: new Date().toISOString(),
    };
  }

  /**
   * Sync single file (Git → Notion)
   * @param {string} filePath - Git file path
   * @param {string} pageId - Notion page ID (optional, lookup from state)
   * @returns {Promise<Object>} Sync result
   */
  async syncGitToNotion(filePath, pageId = null) {
    // Use existing v3.6.2 sync logic
    const manager = new SyncManagerV2(this.notion, { verbose: this.options.verbose });

    const markdown = await fs.readFile(filePath, 'utf-8');

    const result = await manager.sync(markdown, {
      title: path.basename(filePath, '.md'),
      pageId: pageId,
      showProgress: this.options.verbose,
    });

    // Update sync state
    if (result.success && pageId) {
      const commitHash = await this.gitOrchestrator.execGit('rev-parse HEAD');
      this.syncState.updateGitToNotionSync(pageId, commitHash.trim());
    }

    return result;
  }

  /**
   * Sync all tracked pages (Notion → Git)
   * @returns {Promise<Object>} Batch sync results
   */
  async syncAllNotionToGit() {
    const changedPages = await this.changeDetector.detectAllChanges();

    if (changedPages.length === 0) {
      console.log('✅ No changes detected');
      return { success: true, synced: 0 };
    }

    console.log(`🔄 Syncing ${changedPages.length} changed page(s)...\n`);

    const results = [];

    for (const change of changedPages) {
      try {
        const result = await this.syncNotionToGit(change.pageId);
        results.push(result);

        if (result.success) {
          console.log(`✅ ${change.filePath}`);
        } else if (result.skipped) {
          console.log(`⏭️  ${change.filePath} (${result.reason})`);
        }
      } catch (error) {
        console.error(`❌ ${change.filePath}: ${error.message}`);
        results.push({
          success: false,
          pageId: change.pageId,
          filePath: change.filePath,
          error: error.message,
        });
      }
    }

    const successCount = results.filter(r => r.success).length;
    const skipCount = results.filter(r => r.skipped).length;
    const failCount = results.filter(r => !r.success && !r.skipped).length;

    console.log(`\n📊 Sync Summary:`);
    console.log(`   ✅ Success: ${successCount}`);
    console.log(`   ⏭️  Skipped: ${skipCount}`);
    console.log(`   ❌ Failed: ${failCount}`);

    return {
      success: failCount === 0,
      synced: successCount,
      skipped: skipCount,
      failed: failCount,
      results,
    };
  }

  /**
   * Start continuous bidirectional sync
   */
  async startContinuousSync() {
    console.log('🔄 Starting continuous bidirectional sync...');
    console.log(`   Poll interval: ${this.options.pollInterval}ms`);
    console.log(`   Conflict strategy: ${this.options.conflictStrategy}`);

    this.changeDetector.startPolling(async (changedPages) => {
      console.log(`\n🔔 ${changedPages.length} page(s) changed, syncing...`);
      await this.syncAllNotionToGit();
    });

    console.log('✅ Continuous sync started. Press Ctrl+C to stop.');
  }

  /**
   * Stop continuous sync
   */
  stopContinuousSync() {
    this.changeDetector.stopPolling();
    console.log('⏹️  Continuous sync stopped');
  }

  /**
   * Register page for tracking
   * @param {string} pageId - Notion page ID
   * @param {string} filePath - Git file path
   * @param {string} title - Page title
   */
  registerPage(pageId, filePath, title) {
    this.syncState.setMapping(pageId, filePath, title);
    console.log(`✅ Registered ${title} → ${filePath}`);
  }
}
```

**Pros**: Permanent URLs, no Git bloat
**Cons**: Requires S3/CDN setup, costs money

**Recommendation**: Default to Option A (preserve), allow config for B/C

### Challenge 2: Page Links (Notion IDs → Titles)

**Problem**: Notion uses page IDs (`29660b6e...`), markdown uses titles (`[[NotionManager]]`)

**Solution**: Maintain reverse mapping in sync state

```javascript
// Sync state includes page titles
{
  "pageId": "29660b6e8d188144a36ff8857827dc73",
  "title": "NotionManager v3.6.1",
  "filePath": "AGENTS/NotionManager/MEMORY.md"
}

// Converter uses mapping
resolvePageTitle(pageId) {
  const mapping = this.syncState.getMappingByPageId(pageId);
  return mapping ? mapping.title : pageId; // Fallback to ID
}

// Convert mention to wiki link
mentionToMarkdown(mention) {
  if (mention.type === 'page') {
    const title = this.resolvePageTitle(mention.page.id);
    return `[[${title}]]`;
  }
}
```

**Edge Case**: Page not in sync state
- **Fallback**: Fetch page title from Notion API
- **Cache**: Store in sync state for future use

### Challenge 3: Table Formatting Edge Cases

**Problem**: Notion tables can have merged cells, empty cells, inconsistent columns

**Solution**: Normalize table structure

```javascript
normalizeTableRows(rows) {
  // 1. Find max column count
  const maxCols = Math.max(...rows.map(r => r.table_row.cells.length));

  // 2. Pad short rows with empty cells
  return rows.map(row => {
    const cells = row.table_row.cells;
    while (cells.length < maxCols) {
      cells.push([{ type: 'text', text: { content: '' } }]);
    }
    return row;
  });
}

// 3. Ignore merged cells (not supported in markdown)
// Notion API doesn't expose merge info, treat as regular cells
```

**Limitations**: Markdown doesn't support:
- Merged cells
- Cell background colors
- Cell alignment (left/center/right)

**Workaround**: Document in conversion notes, accept data loss

### Challenge 4: Nested Formatting Complexity

**Problem**: Notion allows combinations like bold+italic+code, markdown has precedence rules

**Solution**: Apply formatting in strict order

```javascript
richTextToMarkdown(richTextArray) {
  return richTextArray.map(rt => {
    let text = rt.plain_text;
    const ann = rt.annotations;

    // Priority: code > link > bold > italic > underline > strikethrough > color

    // Code blocks prevent nesting
    if (ann.code) {
      return `\`${text}\``;
    }

    // Links wrap everything
    if (rt.href) {
      // Apply other formats first
      if (ann.bold) text = `**${text}**`;
      if (ann.italic) text = `*${text}*`;
      if (ann.underline) text = `__${text}__`;
      if (ann.strikethrough) text = `~~${text}~~`;

      return `[${text}](${rt.href})`;
    }

    // Regular formats (can nest)
    if (ann.bold) text = `**${text}**`;
    if (ann.italic) text = `*${text}*`;
    if (ann.underline) text = `__${text}__`;
    if (ann.strikethrough) text = `~~${text}~~`;

    // Color (custom tags)
    if (ann.color && ann.color !== 'default') {
      text = `<${ann.color}>${text}</${ann.color}>`;
    }

    return text;
  }).join('');
}
```

**Test Cases**:
```javascript
// Bold + Italic
{ bold: true, italic: true } → ***text***

// Code (prevents nesting)
{ code: true, bold: true } → `text` (bold ignored)

// Link + Bold
{ bold: true, href: 'url' } → [**text**](url)

// All formats
{ bold: true, italic: true, underline: true } → ***__text__***
```

### Challenge 5: API Rate Limiting

**Problem**: Notion API limits 3 requests/second

**Solution**: Batch processing with delays

```javascript
async detectAllChanges() {
  const mappings = this.syncState.getAllMappings();
  const changedPages = [];

  // Process in batches of 10
  for (let i = 0; i < mappings.length; i += 10) {
    const batch = mappings.slice(i, i + 10);

    // Parallel processing within batch
    const results = await Promise.all(
      batch.map(m => this.detectPageChanges(m.pageId))
    );

    changedPages.push(...results.filter(r => r.changed));

    // Rate limiting delay between batches
    if (i + 10 < mappings.length) {
      await this.delay(333); // 1000ms / 3 requests = 333ms
    }
  }

  return changedPages;
}
```

**Calculation**:
- 100 pages = 10 batches × 333ms = 3.3 seconds
- 1000 pages = 100 batches × 333ms = 33 seconds

## 📊 Performance Analysis

### Time Complexity

| Operation | Complexity | Time (100 pages) | Notes |
|-----------|-----------|------------------|-------|
| **Change Detection** | O(n) | ~3.3s | 10 pages/batch, 333ms/batch |
| **Block Conversion** | O(m) | <1s | m = blocks, ~100 blocks/page |
| **Conflict Detection** | O(1) | <10ms | Timestamp comparison |
| **Git Commit** | O(1) | <100ms | Single file operation |
| **Full Sync (100 pages)** | O(n×m) | ~5-10min | n pages × m blocks |

### Memory Usage

```javascript
// Typical memory footprint
NotionToMarkdownConverter:  ~1MB (parser state)
SyncStateManager:           ~100KB per 100 mappings
ChangeDetectionSystem:      ~500KB (polling state)
ConflictResolutionEngine:   ~100KB (diff buffers)

Total:                      ~2-3MB for 100 pages
```

### API Usage

```javascript
// API calls per sync operation
Notion → Git (single page):
  - 1× pages.retrieve (metadata)
  - 1× blocks.children.list (blocks, paginated)
  - n× blocks.children.list (nested blocks)
  Total: 2-5 calls/page

Change Detection (100 pages):
  - 100× pages.retrieve (metadata only)
  Total: 100 calls (~33 seconds with rate limiting)

  - 100× change detection
  - 50× conversion (assuming 50 changed)
  Total: ~200-300 calls (~2-3 minutes)
```

   - **Mitigation**: Strict precedence rules, comprehensive tests
   - **Fallback**: Accept some format loss

2. **Table Structure Variations**
   - **Mitigation**: Normalization logic
   - **Fallback**: Document limitations

   - **Mitigation**: Streaming conversion, pagination
   - **Fallback**: Warn on very large pages

1. **Git Commit Failures**
   - **Mitigation**: Pre-flight checks, atomic operations
   - **Fallback**: Rollback capability

2. **State Corruption**
   - **Mitigation**: Atomic writes, backups
   - **Fallback**: Rebuild from Notion
