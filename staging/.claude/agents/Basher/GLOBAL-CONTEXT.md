### Basher-Specific Guidelines

**When Creating Shell Scripts**:
1. Scripts location: `tools/`, `scripts/`, or appropriate directory
2. Never create scripts in root directory or docs/
3. Script documentation → `working/docs/` then `docs/guides/`

**Anti-Patterns to Avoid**:
- ❌ Creating .sh files in root directory
- ❌ Putting shell scripts in docs/
