# Marketer Agent Sessions

This directory contains records of Marketer Agent sessions, organized by project and marketing focus.

## Session Organization

Sessions are organized into subdirectories based on the marketing project type:

- `ProductMarketing/`: Sessions focused on creating marketing content for products
- `PersonalBranding/`: Sessions dedicated to developing personal and professional branding
- `CorporateMessaging/`: Sessions for company and organizational marketing
- `IdeaPromotion/`: Sessions focused on marketing concepts and ideas
- `ProjectMarketing/`: Sessions for promoting specific projects and initiatives

## Session Structure

Each session directory follows a standard structure:

1. `README.md`: Overview of the marketing project, objectives, and target audience
2. `CONTEXT_ANALYSIS.md`: Assessment of the entity context and competitive landscape
3. `VALUE_PROPOSITION.md`: Articulation of core value and differentiation
4. `MARKETING_CONTENT.md`: The main marketing document in standardized format
5. `FEEDBACK_INTEGRATION.md`: Record of user feedback and iterative improvements

## Format Standards

All primary marketing content follows the UPPERCASE_UPPERCASE.md naming convention, with standardized internal structure:

1. **CHALLENGE_IDENTIFICATION**: The problem or opportunity being addressed
2. **SOLUTION_PRESENTATION**: How the entity solves the challenge
3. **VALUE_DIFFERENTIATION**: What makes this entity uniquely valuable
4. **BENEFIT_ARTICULATION**: Specific advantages for the target audience
5. **EVIDENCE_SUPPORT**: Validation of claims and capabilities
6. **ACTION_DIRECTION**: Clear next steps for the audience

## Usage Guide

When initiating a new marketing session:

1. Provide comprehensive information about the entity to be marketed
2. Specify the target audience and their key motivations
3. Identify the competitive landscape and positioning goals
4. Clarify the primary value proposition to emphasize
5. Indicate any specific voice, tone, or stylistic requirements
6. Describe the intended outcome or call-to-action

## Integration Notes

Session content connects with other agent outputs:

- Designer's visual materials complement Marketer's textual content
- UI/UX insights inform messaging for user interfaces
- Documenter's technical content provides factual foundation
- Human's empathetic understanding enhances emotional appeal
- Expert's domain knowledge strengthens credibility
- Analyst's market research informs competitive positioning
