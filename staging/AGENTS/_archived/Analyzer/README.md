# Analyzer - Smart Analysis & Investigation Routing Agent

## Agent Overview
Analyzer is a smart routing agent that automatically assembles the optimal team of specialists for any analysis, investigation, or strategic assessment task. It intelligently routes requests to the most appropriate analysts based on domain, complexity, and urgency.

## Core Capabilities

### Smart Routing Intelligence
- **Pattern Recognition**: Identifies analysis patterns and selects appropriate specialists
- **Team Assembly**: Automatically coordinates multiple analysts for complex investigations
- **Cross-Domain Analysis**: Routes to specialists across technical, business, and security domains
- **Parallel Investigation**: Manages concurrent analysis streams for faster insights

### Routing Patterns

#### Data Analysis
Routes to: Analyst, Researcher, Scholar
- Statistical analysis and pattern discovery
- Research synthesis and knowledge integration
- Academic and theoretical analysis

#### System Analysis
Routes to: Architect, Topologist, Auditor
- Architecture assessment and design analysis
- Repository and codebase topology mapping
- System auditing and compliance verification

#### Performance Analysis
Routes to: Benchmarker, Optimizer, Profiler
- Performance bottleneck identification
- Optimization opportunity discovery
- Resource utilization analysis

#### Business Analysis
Routes to: StrategicPlanner, BusinessDevelopmentHead, Analyst
- Market analysis and strategic assessment
- Business model evaluation
- ROI and investment analysis

#### Security Analysis
Routes to: Auditor, Cryptographer, Enforcer
- Security vulnerability assessment
- Cryptographic analysis and verification
- Policy enforcement and compliance
