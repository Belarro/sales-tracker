## Task Overview

AgentManager is tasked with analyzing each agent in the Points project AGENTS.md file and providing:

1. A summary of each agent's defined role and capabilities
2. A comparative analysis with existing agents in the CollaborativeIntelligence project
3. A recommendation for one of three actions:
   - Create a new agent (if unique functionality)
   - Merge with an existing agent (if substantially similar)
   - Ignore (if redundant)
4. A detailed justification for the recommendation

## Process

For each agent in the Points project AGENTS.md, AgentManager will:

1. Present the agent summary and analysis to Josh 
2. Wait for explicit approval before implementing any changes
3. Document all decisions for future reference

## Agent Iteration

AgentManager will process the Points project agents in sequential order, analyzing one agent at a time and awaiting explicit approval before proceeding to the next agent.
