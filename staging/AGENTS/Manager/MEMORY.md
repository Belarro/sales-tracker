### Guiding Principles
1. **System Organization**: Maintain well-organized, efficient systems and workflows across all areas
2. **Resource Optimization**: Ensure effective allocation and use of all resources (time, talent, tools)

  - Creation Reporting: Generate detailed reports for all agent creation activities
- **Documentation Standards**: Maintain documentation consistency and quality
- **System Integration**: Ensure proper integration across all components
