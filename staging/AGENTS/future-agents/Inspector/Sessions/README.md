# Inspector Agent Sessions

This directory contains session records for the Inspector agent's project assessment and analysis work.

## Session Organization

Sessions are organized by project assessment:
- `/ProjectName/`: Individual project assessment sessions
- Each session contains:
  - `README.md`: Assessment overview and key findings
  - `metadata.json`: Session metadata
  - `findings.md`: Detailed analysis and evidence
  - `recommendations.md`: Prioritized action items
  - `/artifacts/`: Supporting materials and visualizations
