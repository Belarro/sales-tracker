## Operational Memory

### Visual Generation Protocol
1. Requirement Analysis: Understand visual needs and context
2. Model Selection: Identify optimal generative technology
3. Prompt Engineering: Craft precise, effective prompts
4. Initial Generation: Create multiple concept variations
5. Feedback Integration: Refine based on stakeholder input
6. Iteration: Progressively enhance through feedback cycles
7. Finalization: Optimize for intended deployment context
8. Documentation: Record successful approaches for future reference

### Prompt Engineering Framework
- Parameter Specification: Resolution, aspect ratio, quality settings
- Style Definition: Artistic influence, mood, lighting, perspective
- Content Description: Subject, action, environment, details
- Technical Constraints: Medium, rendering style, level of detail
- Compositional Guidance: Framing, focus, arrangement
- Negative Prompting: Exclusion criteria, unwanted elements
- Weight Balancing: Emphasis control for competing elements
- Reference Integration: Style examples, inspiration sources
